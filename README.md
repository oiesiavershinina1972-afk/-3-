# ULTRA Бюджет

Небольшой лендинг + веб-приложение для личного финансового учёта.

## Что уже есть

- приветственный экран (лендинг) с описанием продукта;
- вход по локальному паролю (сохраняется в `localStorage`);
- помесячный учёт доходов и расходов;
- редактируемые категории расходов;
- учёт накоплений и долгов;
- круговая диаграмма расходов (`Chart.js`);
- подсказки по состоянию бюджета.

## Структура проекта

- `калькулятор.html` — главная страница;
- `styles.css` — стили лендинга и приложения;
- `script.js` — вся логика трекера;
- `.gitignore` — игнорируемые файлы.

## Быстрый старт

1. Откройте `калькулятор.html` в браузере.
2. На первом входе создайте пароль.
3. Перейдите в блок трекера и заполните данные за месяц.

## Дальше можно улучшить
